## 🚀 Quick Start

```bash
# 1. Setup environment
cp server/.env.example server/.env
cp client/.env.example client/.env

# 2. Install dependencies and seed database
npm run setup

# 3. Start development servers
npm run dev

# 4. Verify integration
npm run verify
```
